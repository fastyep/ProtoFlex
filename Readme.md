# ProtoFlex

```
USAGE: flex [--help] [--inputfile <path>] [--outputfile <path>] [--namespace <name>]

OPTIONS:

    --inputfile, -i <path>
                          path to proto files to build.
    --outputfile, -o <path>
                          path where to write final FSharp types bundle.
    --namespace, -n <name>
                          namespace that will be used in final bundle file.
    --help                display this list of options.
```
